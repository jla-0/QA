## Local Git Commands

Run these from `C:\WritingPortfolio\QA_Manual_Tester` when you are ready to publish:

```powershell
git init
git add .
git commit -m "Create healthcare claims manual QA portfolio"
git branch -M main
git remote add origin https://github.com/jla-0/QA.git
git push -u origin main
```
