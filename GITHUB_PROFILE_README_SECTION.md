# Featured Project: Healthcare Claims Manual QA Portfolio

This synthetic healthcare claims QA portfolio demonstrates manual SQA work in a regulated healthcare environment.

The project includes:

- requirements analysis and QA strategy;
- test plan and traceability matrix;
- manual test cases for claim intake, eligibility, provider contract validation, adjudication, denial handling, payment, and claim status;
- SQL backend validation queries;
- SOAP/XML claim status request and response samples;
- EDI-style 837 and 835 examples;
- Jira-ready defect writeups;
- HIPAA-safe synthetic data strategy;
- regression, integration, and load testing planning.

Project folder: `healthcare-claims-manual-qa-portfolio`
